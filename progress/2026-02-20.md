# Progress — 2026-02-20

## Task: Manual Origin Homing Strategy for Genmitsu Desktop (CUB)

### Summary
Implemented the `manual_origin` homing strategy for the Genmitsu 3018 PRO Desktop CNC. Instead of seeking limit switches, the user interactively jogs the gantry to the desired origin using keyboard controls, presses Enter to confirm, and the system sets that position as work coordinate zero (`G10 L20 P1 X0 Y0 Z0`).

### How It Works
1. `Gantry.home()` checks `config["cnc"]["homing_strategy"]`
2. When `"manual_origin"`, dispatches to `Mill.home_manual_origin()`
3. The method enters an interactive keyboard loop (reuses `setup/keyboard_input.py`)
4. User jogs with arrow keys (X/Y) and Z/X keys (Z axis)
5. On Enter: sends `G10 L20 P1 X0 Y0 Z0` to set current position as work zero
6. Sets `self.homed = True`
